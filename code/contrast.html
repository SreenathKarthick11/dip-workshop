Contrast Adjustment

Theory

Contrast adjustment enhances differences between light and dark areas of an image. This can be achieved using linear transformations or scaling pixel intensities.

Python Code


import cv2
import matplotlib.pyplot as plt

img = cv2.imread("assets/sample.jpg", 0)
alpha = 1.5   # contrast control
beta = 0      # brightness shift
adjusted = cv2.convertScaleAbs(img, alpha=alpha, beta=beta)

plt.subplot(121), plt.imshow(img, cmap='gray'), plt.title("Original")
plt.subplot(122), plt.imshow(adjusted, cmap='gray'), plt.title("Contrast Enhanced")
plt.show()
