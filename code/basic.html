Basic Image Operations

Theory

This example introduces fundamental image operations in OpenCV:

  • Loading an image in different modes (color, grayscale, unchanged).
  • Checking image shape (rows, columns, color channels).
  • Resizing an image using absolute pixel size or scale factors.
  • Rotating an image (90Â° clockwise in this example).
  • Displaying images in separate windows.
  • Saving processed images to disk.

Python Code


import cv2
import numpy as np

# Load an image
# -1 or cv2.IMREAD_COLOR     : Load a color image (default)
#  0 or cv2.IMREAD_GRAYSCALE : Load in grayscale mode
#  1 or cv2.IMREAD_UNCHANGED : Load including alpha channel
img = cv2.imread("assets/bear.jpg")

print("Image shape:", img.shape)   # (rows, cols, channels)

# Convert to grayscale
gray_image = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Resize the image (two methods)
# Method 1: Absolute size
# img = cv2.resize(img, (400, 400))  

# Method 2: Relative scaling (50% of original size)
img = cv2.resize(img, (0, 0), fx=0.5, fy=0.5)

# Rotate the image (90 degrees clockwise)
img = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)

# Display images
cv2.imshow("Original (Rotated + Resized)", img)
cv2.imshow("Grayscale", gray_image)

# Save image (optional)
# cv2.imwrite("bear_processed.jpg", img)

# Wait until any key is pressed, then close windows
cv2.waitKey(0)
cv2.destroyAllWindows()
